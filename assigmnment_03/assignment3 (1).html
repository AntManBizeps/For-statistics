1. Linear Regression Model for Wine Quality Prediction (8+6)

You have 2 datasets containing information about red and white wines, including various attributes such as acidity, alcohol content, and residual sugar. The goal is to predict the quality of the wine based on these features using a linear regression model.

a) Dataset preparation (2+0)

  1. Implement the function read_data that takes the file paths of the red and white wine datasets as arguments. The function should read the datasets, add a new column named type to each DataFrame (with value 0 for red wine and 1 for white wine), and then merge the two DataFrames into a single dataset. Finally, the function should return the merged dataset.

b) Exploratory Data Analysis (1+2)
To get a better overview of the data, plot the distribution (histogram) of each input feature and the target variable (quality) in the merged dataset and include it in the report.

IMPORTANT: Keep the plots relatively small (maximal one page for all 12 subplots).

Also describe and interpret at least 4 of the 12 histograms in 1 or 2 sentences. Note any interesting patterns or observations.

c) Cross-Validated Linear Regression Model (4+2)

Consider fitting a linear regression model to predict wine quality. Utilize a 5-fold cross-validation with an 80/20 train-test split. Briefly explain the significance of cross-validation in model evaluation.

  • Standardize the features by removing the mean and scaling to unit variance
  • Perform the 5-fold cross-validation, for each fold:
    • Split the data (80/20)
    • Fit the model
    • Predict the data
    • Store R2 metric

Hint: Use the sklearn python library for all the sub tasks like: train-test splitting, linear regression, cross-validation, model-fitting, model-prediction.

d) Metrics Calculation and Interpretation (1+2)

Now, to get a understanding how well our model predicts the wine quality, provide an interpretation of your results in the report:

  • Cross-validation scores (individual R2 for each fold)
  • Averaged cross-validation score (mean value of R2)

Discuss why R2 is a meaningful metric for evaluating the performance of regression models. Determine the practical implications of a high or low R2 in the context of wine quality prediction.

2. Model Selection with Greedy Search (4+6)

In this task, we try to optimize the linear regression model for wine quality prediction by implementing a greedy backward search functionality for feature selection. The goal is to drop low influence features from the full model.

a) Feature Selection using Backward Search (4+3)

Optimize the full model from Task 1 by conducting a greedy backward search in order to eliminate features. Use the full model with all features as a starting point. Your implemntation should cover the following functionality:

  • Iteratively remove features from the model
  • At each iteration:
    • Apply 5-fold cross validation with the reduced feature set
    • Drop the least performing feature based on R2 (use the average of the folds)
    • Record the dropped feature (needed for the report)
  • Return/save the best model (needed for b) Model comparison)

Provide a list containing the optimal set of features selected for your optimized model. Furthermore, provide a list or ranking of the features dropped during each iteration, indicating which feature was removed in each respective iteration. (Report)

Hint: Try to reuse/adapt your functions from Task 1.

b) Model comparison (0+3)

Compare the performance of the full model (from Task 1) with the optimized model obtained through backward feature selection. Evaluate both models by using the metric R2. Discuss and analyze any notable differences in performance between both models. Additionally, provide an interpretation of the impact of feature elimination on the model’s efficiency.

3: Logistic Regression Model for Wine Type (8+3)

In this task you extend the wine analysis by implementing a logistic regression for the wine type.

a) Logistic Regression for Wine Type (5+2)

By comparing the values of the different features between red and white wine, we observe that it might be possible to predict the wine type based on the feature values of a wine. Use logistic regression with the additional categorical feature, type, which you added in Task 1, indicating if it is a red or white wine.

Implement the following steps for logistic regression:

  • Apply 5-fold cross-validation with an 80/20 split to the logistic regression model with the added type feature. Use a threshold of t=0.5 to determine the predicted type. A value t<0.5 shall be red wine, otherwise white wine.
  • Calculate the mean accuracy over all folds: accuracy=TP+TNTP+TN+FP+FN, where TP is the number of true positives, TN the number of true negatives, FP the number of false positives and FN the number of false negatives.
  • Report the achieved mean accuracy and create a confusion matrix heatmap plot for each fold, containing TP,TN,FP and FN. Explain how logistic regression differs from linear regression!

b) Threshold fine-tuning (3+1)
You are curious if it is possible to improve the model further and set your eyes on the threshold value. You found out, that increasing or decreasing the threshold influences the accuracy of the model:

  • Iteratively test thresholds in steps of s=0.05 in the range of Rt=(0,1) and calculate the mean accuracy for each threshold (5-fold cross-validation).
  • Create a line plot that shows the mean accuracy as the y value and the threshold (only label each 0.1 step) as x. What can you observe?
  • Report the threshold and mean accuracy of the best result!

Hint: You are allowed to reuse the code from Task a. Use the sklearn python library for all the sub tasks like: train-test splitting, logistic regression, cross-validation, model-fitting, model-prediction. You are allowed to use metric functions from sklearn.metrics
